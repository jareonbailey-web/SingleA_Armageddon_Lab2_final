or	
https://github.com/jareonbailey-web/20jan26_class_git.git
Get started by creating a new file or uploading an existing file. We recommend every repository include a README, LICENSE, and .gitignore.

create a new repository on the command line
git init
git add README.md
git commit -m "first commit" (add message explaining the reason for the action)
git branch -M main
git remote add origin https://github.com/jareonbailey-web/20jan26_class_git.git (change repo folder)
git push -u origin main

push an existing repository from the command line
git remote add origin https://github.com/jareonbailey-web/20jan26_class_git.git
git branch -M main
git push -u origin main
